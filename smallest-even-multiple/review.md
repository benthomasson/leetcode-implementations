# Review: Smallest Even Multiple

## Solution
- **Approach**: Returns n if even (already divisible by 2), otherwise returns n*2 (smallest common multiple when n is odd).
- **Time Complexity**: O(1)
- **Space Complexity**: O(1)
- **Correctness**: Correct
