# Review: Missing Number

## Solution
- **Approach**: Uses Gauss's formula to compute expected sum of range [0, n], then subtracts actual sum to find the missing number.
- **Time Complexity**: O(n)
- **Space Complexity**: O(1)
- **Correctness**: Correct

- **Edge Cases**: Yes, covers missing 0, missing n, single element arrays, and larger arrays
